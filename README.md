# Conexion via Bluetooth a Raspberry pi zero w con OBD2 Bluetooth
Este proyecto Busca dejar evidencia de un procedimiento realizado con <br/>  
un raspberry pi y un dispositivo OBD2 Bluetooth con el fin de lograr  <br/> 
conexion y luego lectura de datos de un Vehiculo.

Con la Generacion de un puerto serie se realiza la comunicacion con   <br/> 
el dispositivo OBD-ii ELM327 Bluetooth al Raspberry pi Zero W.
